### 📋 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:

* [Git](https://git-scm.com/downloads "Download Git") deve estar instalado em seu sistema operacional.

### 📍 Executar localmente

Para executar o repositório localmente, execute este comando no seu git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/p1nheiros/numeros-primos.git
```

Windows:

```bash
git clone https://github.com/p1nheiros/numeros-primos.git
```
